# terraform_aws_vpc
this example covers the following topics:
  1. create a vpc
  2. create dhcp options
  3. create internet gateway
  4. create subnets
  5. create customized route tables and associate them to subnets
  6. create nat gateway and associated to route table
  7. create network acl and associated to subnets
  8. create security groups
  9. create prefix lists and use them in security groups
